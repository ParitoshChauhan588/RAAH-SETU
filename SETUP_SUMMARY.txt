â“ TROUBLESHOOTING:

Problem: "Python not found" or "python command not recognized"
Solution: 
  1. Install Python from https://www.python.org/
  2. During install, CHECK "Add Python to PATH"
  3. Restart PowerShell
  4. Try: python --version

Problem: "npm not found" or "npm command not recognized"
Solution:
  1. Install Node.js from https://nodejs.org/
  2. Restart PowerShell
  3. Try: npm --version

Problem: "Port 5000 or 8080 already in use"
Solution:
  1. Find the process: netstat -ano | findstr :5000
  2. Kill it: taskkill /PID <ProcessID> /F
  3. Try running again

Problem: "Can't connect to database"
Solution:
  1. MySQL may not be running
  2. Install from: https://dev.mysql.com/downloads/mysql/
  3. Or skip for now (uses default SQLite backup in code)

Problem: "Module not found" (pip install fails)
Solution:
  1. Make sure you're in correct directory: cd Backend
  2. Upgrade pip: python -m pip install --upgrade pip
  3. Clear cache: pip cache purge
  4. Try again: pip install -r requirements.txt
